# 产品需求规格说明书

## 一、项目概述

### 1.1 项目名称
AI-Powered Stock Research Assistant（人工智能股票研究助手）

### 1.2 项目定位
一款面向股票行业研究员的 AI 自动化投研工具，把传统人工繁琐的资料搜集、信息整理、竞品对标、论点梳理工作自动化，输出可直接用于 PPT 演示的标准化投研简报。

### 1.3 目标用户
股票行业研究员、投资分析师，面向办公桌面端大屏使用。

### 1.4 运行环境
- 前端：PC 端网页浏览器（Chrome / Edge 主流桌面浏览器）
- 后端：Web 服务端部署，浏览器直接访问网页地址即可使用
- 底层依赖：LLM 大模型接口 + 公开财经数据源接口

---

## 二、行业痛点

传统股票基本面投研工作中，研究员的核心瓶颈并非最终投资判断，而是大量重复性体力工作：

1. **资料搜集耗时**：手动搜集财报、业绩发布会文稿、公司新闻、竞品财报等海量零散资料，耗时长达数天
2. **信息提炼低效**：人工从长文稿里提炼管理层核心观点、季度财务变动、潜在业务风险，效率极低
3. **横向对标繁琐**：手动横向对标多家同行公司的利润率、增速、研发投入、估值等多维度数据
4. **报告整理机械**：手动整理多空投资逻辑，排版梳理成结构完整的投研汇报 PPT 初稿

本软件依靠 AI 流水线 + 模块化代码架构，将资料收集与结构化整理的周期从数天压缩至数小时。

## 三、核心功能模块

### 功能 1：全自动财经数据采集
输入目标股票标的后，自动抓取：
- 财报公告（季报/年报）
- 投资者演示文稿（Investor Presentation）
- 业绩电话会议完整文稿（Earnings Call Transcript）
- 官方新闻稿（Press Releases）
- 同行业竞品全套财务数据

### 功能 2：文档智能解析结构化提炼
调用大模型对海量长文档做智能阅读，自动结构化提炼：
- 季度业务核心变动
- 管理层战略重心
- 财务数据异动（同比/环比异常变动）
- 业务风险预警
- 待确认业务疑问

### 功能 3：同业横向对标分析
围绕以下维度完成标的公司与同行竞品的完整多维度对标：
- 营收增速（YoY/QoQ）
- 毛利率 & 净利率
- 研发投入强度（R&D as % of Revenue）
- 客户结构
- 业务落地进度
- 估值水平（PE/PS/PB/EV-EBITDA）
- 业务叙事差异化

### 功能 4：多空均衡投资论点生成
自动生成平衡客观的完整投资逻辑：
- 上涨驱动催化（Catalysts）
- 市场被低估的潜在利好（Underappreciated Positives）
- 下行潜在风险（Downside Risks）
- 当前股价已经定价的市场预期（Priced-in Expectations）

### 功能 5：PPT 级标准化简报一键生成
输出可直接复用修改的幻灯片格式投研简报，固定模块：
1. 一页核心投资论点（Executive Summary）
2. 公司业务概览（Company Overview）
3. 财务数据快照（Financial Snapshot）
4. 多头逻辑（Bull Case）
5. 空头逻辑（Bear Case）
6. 催化因素与风险总结（Catalysts & Risks）
7. 最终投研结论（Conclusion）

**所有产出内容附带原文溯源引用标注，保证信息可追溯。**

## 四、页面结构与功能

### 页面 1：首页 / 项目新建工作台
| 模块 | 内容 |
|------|------|
| 标的搜索框 | 输入股票代码或名称搜索目标公司 |
| 竞品添加栏 | 批量添加同业对标公司 |
| 数据源筛选 | 财报 / 新闻 / 电话会 / 投资者演示 开关筛选 |
| 启动按钮 | 一键启动全自动数据采集流程 |
| 历史项目列表 | 展示已创建的研究项目，支持重新打开/删除 |

### 页面 2：数据采集进度页
| 模块 | 内容 |
|------|------|
| 进度概览 | 分模块（财报/新闻/电话会/其他）实时展示抓取进度 |
| 已完成文档列表 | 已成功抓取的文档清单，可点击预览 |
| 缺失数据源提示 | 标注未能自动获取的数据，提示用户补充 |
| 手动上传入口 | 支持拖拽上传本地 PDF/Word/文本 内部文档 |

### 页面 3：文档解析 & 信息提炼工作台
| 面板 | 内容 |
|------|------|
| 左侧 | 原始文档列表，点击切换查看原文 |
| 中间 | AI 提炼的结构化核心信息（业务变动 / 财务异动 / 管理层讲话 / 风险点），每条附带来源引用标注 |
| 右侧 | 研究员编辑面板，支持对 AI 提炼内容进行修改/补充/删除 |

### 页面 4：同业对标分析页面
| 模块 | 内容 |
|------|------|
| 对标表格 | 横向罗列标的公司与多家竞品在多维度的数据对比 |
| 可视化图表 | 简易柱状图/雷达图对比关键指标 |
| 高亮标注 | 显著差异项自动标色（优于同业绿色/劣于同业橙色） |

### 页面 5：多空投资逻辑梳理页
| 板块 | 内容 |
|------|------|
| 多头利好催化 | 上涨驱动论点列表，每条附带数据支撑与来源溯源 |
| 空头下行风险 | 下行风险论点列表，每条附带数据支撑与来源溯源 |
| 自定义观点 | 支持研究员增删修改自定义论点 |
| 操作按钮 | 添加论点 / 编辑 / 删除 / 重新生成 |

### 页面 6：最终简报预览 & 导出页
| 模块 | 内容 |
|------|------|
| 简报预览 | 完整幻灯片版式预览，分页展示整套汇报结构 |
| 页面导航 | 幻灯片缩略图侧栏，点击切换预览 |
| 导出功能 | 导出 PDF（直接下载）/ 导出可编辑 PPT 文件 |

## 五、设计规范

### 5.1 设计风格
极简专业商务办公风，偏向金融投研工具的严谨克制风格，无多余花哨动效，页面层级清晰，功能分区明确。以高效办公效率为优先，信息密度合理，研究员可以快速定位功能模块。偏向投行/资管内部专业 SaaS 工具设计质感。

### 5.2 核心配色
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #163D7A | 深海蓝，金融专业稳重感 |
| 辅助色 1 | #E8EFF9 | 浅灰蓝，页面分区底色 |
| 辅助色 2 | #E6772E | 警示橙，风险数据、空头论点高亮 |
| 辅助色 3 | #2D995F | 增长绿，利好增长、多头催化高亮 |
| 基础底色 | #FFFFFF | 纯白 |
| 正文文字 | #333333 | 中性深灰，长时间阅读护眼 |

### 5.3 细节规范
- 表格数据采用明暗交替行底色
- 重要异动数据自动标色区分（正异动用绿、负异动用橙）
- 所有引用来源采用浅灰色上标标注
- 引用标注可点击快速跳转回原始文档片段
- 字体：系统默认中文字体（微软雅黑 / PingFang SC），英文数字等宽字体

## 六、非功能性需求

### 6.1 性能
- 页面首次加载时间 < 3 秒
- 数据采集异步进行，不阻塞 UI 交互
- API 响应时间（非 AI 处理类）< 500ms

### 6.2 安全
- 用户密码加密存储（bcrypt）
- API 接口 JWT 认证
- 用户数据隔离（只能访问自己的项目）

### 6.3 兼容性
- 支持 Chrome 最新两个大版本
- 支持 Edge 最新两个大版本
- 最低屏幕分辨率：1366 × 768
- 推荐屏幕分辨率：1920 × 1080 及以上

### 6.4 可维护性
- 数据源接口模块化，可独立替换
- LLM 接口模块化，支持切换不同模型
- 前后端分离架构，独立部署和扩展
